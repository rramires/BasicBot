# BasicBot
Bot básico para Binance, com o objetivo de aprendizado da API
***
### Códigos de erro comuns na API da Binance:
#### 400 - Passou algum parâmetro errado, ou fora de ordem 
#### 401 - Chaves(API Key) com problema 
#### 404 - URL Errada
